# java-rpg-game

# Overview
This project is a console-based Role-Playing Game (RPG) developed in Java. Players battle enemies, manage health, and progress through combat encounters while demonstrating object-oriented programming principles.

# Features:
~Player name validation

~Random enemy generation

~Turn-based combat

~Basic Strike

~Prime Strike

~Modulus Strike

~Health and score tracking

~Console-based gameplay

# Technology used:
~Java

~Object-Oriented-Programming

~NetBeans

~Scanner

~Random

# Concept Demonstrated:

Classes and Objects

~Inheritance

~Polymorphism

~Abstract Classes

~Conditional Statements

~Loops

~Methods

~Input Validation
